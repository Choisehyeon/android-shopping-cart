# android-shopping-cart

### 기능 목록

- [ ] 상품을 클릭하면 상세 페이지로 이동한다.
- [ ] 상품 상세에서 장바구니에 상품을 담을 수 있다.
- [ ] 장바구니에서 원하는 상품을 삭제할 수 있다.
- [ ] 최근 본 상품 목록을 가져온다.
  - [ ] 목록 상단에서 10개까지 확인할 수 있다.
- [ ] 앱이 종료돼도 최근 본 상품 목록과 장바구니 데이터는 유지돼야 한다.

- [x] Product
  - id, 이름, 금액, 사진을 가짐

- [ ] Cart
  - [ ] 상품을 담을 수 있다.
  - [ ] 상품을 뺄 수 있다.
  - [ ] 상품을 불러올 수 있다.
  
- [ ] RecentProduct(linkedlist)
  - [ ] 최근 본 상품이 비어있는지 확인
  - [ ] 상품을 담을 수 있다.
  - [ ] 상품을 불러올 수 있다.
  - [ ] 중복이 있는 경우 원래 있는 애를 제거한다.
